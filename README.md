# React + Tailwind E-Commerce Website

## 🛍️ Description
A modern and fully responsive e-commerce website built with **React.js** and **Tailwind CSS**.  
This project comes with product categories, add-to-cart functionality, checkout flow, and a sleek UI design.

## ✨ Features
- 🛒 Add to Cart & Remove from Cart
- 🔍 Product Filtering & Sorting
- 🌗 Dark/Light Mode Toggle
- ✅ Fake Checkout Page
- 📱 Fully Responsive Design
- 🎨 Modern UI using Tailwind CSS

## 🛠️ Tech Stack
- **React.js** (Frontend)
- **Tailwind CSS** (Styling)
- **React Hooks** (`useState`, `useEffect`)
- **LocalStorage** (for saving cart items)

## 🚀 How to Run
1. Download or clone the project
2. Open terminal in the project folder
3. Run the following commands:
   ```bash
   npm install
   npm run dev
